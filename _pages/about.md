# 💻 Research and Project Experience

## Olympic Medal Prediction and Mechanism Quantification

*Research-guided project, Member*  
*2025.01 - 2025.12*

This project focuses on Olympic medal prediction and mechanism analysis based on multi-dimensional data. To address the nonlinear distribution of medals and the zero-inflation problem in predictions for small countries, we proposed a framework combining random forest models and Word2Vec-based semantic embeddings.

My contributions include:

- Introduced Word2Vec semantic embeddings to model latent physiological correlations across sports.
- Built a logistic regression classifier for first-gold-medal prediction for small countries.
- Assisted in statistical analysis of the host-country effect and elite-coach effect.
- Contributed to sensitivity analysis and manuscript preparation.

The model achieved a test RMSE of **7.877**, and the AUC for first-gold-medal prediction reached **0.97**.

## EEG-based Hallucination Detection for Large Language Models

*Research training project & National Undergraduate Innovation Project, Member*  
*2026.01 - Present*

This project aims to detect hallucinations in large language models by incorporating real-time human cognitive feedback. We designed a dual-brain EEG hyperscanning paradigm and developed a deep learning framework for token-level hallucination warning.

My contributions include:

- Assisted in the implementation of the dual-brain hyperscanning experimental paradigm.
- Participated in the construction of the core EEG dataset.
- Completed synchronous EEG data collection from **40 participants**.
- Conducted ICA-based denoising, artifact removal, and token-level timestamp alignment.
- Assisted in result analysis and paper preparation.

## LLM-based Interactive Music Recommendation System

*University-level Undergraduate Innovation Project, Project Leader*  
*2025.03 - 2026.04*

This project focuses on building an interactive music recommendation system based on large language models. To reduce the information-cocoon effect in traditional recommendation systems, we introduced the BiLLP framework and separated the recommendation process into high-level emotional planning and song-level execution.

My contributions include:

- Led the overall system design and project implementation.
- Developed core prompts for multi-round interaction.
- Integrated cloud-based LLM APIs into the recommendation pipeline.
- Built a web prototype using Gradio.
- Wrote the final technical report as the first contributor.

## MLS Match Database and Dashboard System

*Course Project, Team Leader*  
*2025.05 - 2025.06*

This project involved the design and implementation of a relational database and dashboard system for Major League Soccer match data. The dataset covered **29 teams** and more than **2,000 players**, collected from real-world football data sources.

My contributions include:

- Designed an E-R model with 8 core entities.
- Implemented primary-key and foreign-key constraints.
- Wrote complex SQL queries involving nested JOINs and UNION ALL.
- Built advanced views such as attack index and season standings.
- Delivered a data reporting system for tactical and business analysis.
